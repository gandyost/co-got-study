[8일차] 박스 모델 살펴보기

h1태그: 화면 전체를 사용하고 있음
a태그: 줄바꿈을 하지 않고 딱 자기 콘텐츠 크기만큼만 화면을 쓴다.

화면 전체를 사용하는 태그를 block level element, 자기 자신 만큼의 크기를 사용하는 태그를 inline element라고 한다.

만일 제목 태그들을 화면 전체를 사용하지 않고, 자기 자신만큼만 쓰게 하고 싶다면 style 태그 안에 display:inline 으로 꾸며주면 된다.

이렇게 말이다.

<= 이 부분은 h2태그를 달았으나, 화면 전체를 사용하지 않는다.
반대로, a태그는 display:block 스타일을 적용하여 화면 전체를 사용하게 만들 수 있다.

만일 특정 태그를 안 보이게 하고 싶다면 display:none 을 사용하면 화면에서 사라지게 된다.


컨텐츠와 박스 사이에 간격을 주고 싶다면 padding을 사용한다. 박스 사이의 간격은 margin으로 조정한다.
width를 사용하면 화면 전체를 차지하는 박스의 크기를 임의로 조절할 수 있다. 예를 들어, width: 50px와 같은 형태로 써주면 해당 픽셀 크기만큼 박스 크기를 바꿀 수 있다.


웹페이지에서 마우스 오른쪽 버튼을 누르면 나오는 메뉴 중 '검사'라는 항목이 있다. 검사를 눌러보면 해당 페이지의 html 코드를 확인할 수 있다. styles 탭을 활용하면 나의 웹페이지가 어떤 코드의 영향을 받고 있는지 확인할 수 있다. 이 기능을 사용하면 어떤 하나의 태그가 어떤 CSS의 영향을 받고 있는지 알 수 있다. '검사' 도구를 잘 활용하여 웹페이지를 분석할 수 있어야 한다.
