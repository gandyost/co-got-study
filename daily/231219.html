목차 작성하기, 부모 태그와 자식 태그 살펴보기

  1. HTML
  2. CSS
  3. JavaScript
목차(목록)를 만들라고 준 태그: 리스트, li
서로 다른 목차끼리 구분을 해주기 위한 태그: ul(unordered list) 태그. 서로 연관된 목차가 어디서부터 어디까지인가를 나타내줌.
이 경우, li 태그는 자식 태그, ul 태그는 부모 태그

목차에서 넘버링을 할 때, 1. 2. ... 하나하나 직접 입력하면 중간에 수정사항이 발생할 때 처음부터 다 고쳐야 하는 상황이 발생한다. 이러한 문제를 해결해줄 수 있는 태그가 ol(ordered list) 태그이다.

문서의 구조와 슈퍼스타들

빈도수가 압도적으로 높은 슈퍼스타 태그 살펴보기

1. 타이틀 태그: 웹페이지의 이름을 명시적으로 알려주기. 타이틀 태그를 쓰면 html 문서의 제목이 아니라 내가 원하는 제목으로 만들 수 있다.
2. 원래 한글은 utf-8 형식으로 웹페이지를 열어야 깨지지 않음. 해당 형식으로 문서를 여는 태그: meta charset="utf-8"
3. 문서를 설명하는 태그는 head 태그, 본문은 body 태그로 묶어주기로 약속함
4. head와 body 태그는 모두 html 태그 안에 있도록 가장 바깥에 html 태그를 사용해주어야 한다.
5. 문서 맨 위에는 이것이 html 형식 문서라는 것을 알리는 태그를 달아주어야 한다.
