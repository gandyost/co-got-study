CSS 개요 정리하기

CSS란?
  • 웹페이지의 디자인을 위해 개발된 언어이다.
  • 동일한 속성을 가진 내용을 한 번에 꾸미기 위한 코드이다.
  • 헤드 태그 안에 스타일 태그를 사용하여 해당 부분이 CSS 코드임을 알 수 있다.
  • 본문 태그 안에서는 스타일을 직접 입력하여 스타일을 바꾸어줄 수 있다.
  • 꾸밈효과를 서로 구분해주기 위해서 각 꾸밈효과 끝에 세미콜론(;)을 적어주어야 한다.
